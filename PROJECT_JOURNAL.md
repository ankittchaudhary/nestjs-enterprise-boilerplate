### Understanding main.ts

- main.ts is the entry point of a NestJS application.
- bootstrap() starts the application.
- NestFactory.create() creates and initializes the Nest application from the root AppModule.
- await ensures the application is fully initialized before continuing.
- app.listen() opens a port and waits for incoming HTTP requests.
